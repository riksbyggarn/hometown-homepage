# Hometown Homepage

A simple, responsive homepage for Gothenburg, Sweden. Showcases the city's highlights and your local guide!

## Demo

[View live!](https://gothenburgguide.netlify.app/)

## Features

- Hero section with background image and city description
- Top three activities in Gothenburg, each with image and description
- Local guide section with photo and personal message
- Responsive design for desktop and mobile

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)

## Installation

1. Clone the repo:
   `git clone https://github.com/riksbyggarn/hometown-homepage.git`
2. Open `index.html` in your browser.

## Usage

- Explore the homepage to discover Gothenburg's highlights and activities.
- View on desktop or mobile for a responsive experience.
